# ClaudeChef

**AI-powered recipe creation from your ingredients!**  
Turn what's in your kitchen into something delicious with ClaudeChef.

[Visit the App](https://myclaudechef.netlify.app/)

## Overview

ClaudeChef is an AI-powered recipe generator built using **React**, **Netlify**, and **OpenRouter API**, powered by models like **DeepSeek** and **Qwen**.

Users input the ingredients they have (e.g. *eggs, flour, tomatoes*), and ClaudeChef responds with a complete recipeâ€”title, ingredients, steps, and tips.

## Usage

1. Enter ingredients in the input field.
2. Click the **"Generate Recipe"** button.
3. Wait a moment as AI prepares a recipe.
4. Cook or try again with different ingredients!
